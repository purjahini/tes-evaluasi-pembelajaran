# tes-evaluasi-pembelajaran
tes online pilihan ganda dengan tujuan mengevaluasi hasil belajar siswa

Tes/Ujian Online ini berfokus pada proses analisa hasil tes para siswa dengan menggunakan analisa butir soal diantaranya mencari daya pembeda dan tingkat kesukaran soal.
Pembuatan Tes Online ini bertujuan agar memudahkan guru dalam mengevaluasi hasil belajar mengajarnya, dengan aplikasi ini berharap dapat memudahkan guru dalam mengetahui siswa mana yang sudah paham atau yang belum terhadap pelajaran yang telah diajarkan. Saat pembuatan aplikasi ini hanya dikuhusukan untuk sekolah tingkat pertama karena pilihan ganda nya hanya empat saja belum bisa dinamis.


# FITUR:
* CRUD Data
* Menampilkan hasil analisa

# Bugs
